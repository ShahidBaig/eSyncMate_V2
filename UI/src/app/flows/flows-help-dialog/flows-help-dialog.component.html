label Understanding Flow Status

Each flow has a status that tells you whether it's currently running or paused:

check_circleActive

The flow is live and running on schedule. All active routes within this flow will execute automatically at their configured times.

pause_circleIn-Active

The flow is paused. No routes will run until the flow is reactivated. Existing schedules are preserved but not executed.

touch_app What You Can Do

These action buttons let you manage your partner flows:

add
Add Partner Flow

Create a new flow for the selected partner. Configure the flow name, description, and add one or more data routes with scheduling options.

visibility
View Flow

Open a read-only view of the flow showing route details, a visual flow diagram, and the hub-and-spoke data map. You can also trigger a Test Run for any active route.

edit
Edit Flow

Modify the flow name, description, status, or update route details and their schedules. Changes take effect on the next scheduled run.

delete
Delete Flow

Remove the flow and all its scheduled jobs. A confirmation dialog will show you how many routes will be affected before you confirm.

schedule Setting Up Schedules

Each route in a flow can be scheduled to run at different intervals:

timer
Minutely Runs every few minutes based on the repeat count (e.g., every 5 minutes)
hourglass_top
Hourly Runs every few hours, optionally at specific times you choose
today
Daily Runs every day at the specific times you set (e.g., 8:00 AM, 2:30 PM)
date_range
Weekly Runs on selected days of the week (Mon, Tue, Wed, etc.) at your chosen times
calendar_month
Monthly Runs on specific days of the month (e.g., 1st and 15th) at your chosen times
lightbulb Each route can only belong to one flow. The schedule determines when data exchanges happen automatically.

preview Inside the View Flow Dialog

When you click View Flow, you'll see three tabs:

list_alt Route Details

Shows each route's frequency, start date, next run time, and current status. Active routes have a Test Run button for immediate one-time execution.

schema Flow Diagram

A visual pipeline showing Source, Route, and Destination for each step. You can download the diagram as PDF or expand it to fullscreen.

hub Hub & Spoke

A hub-and-spoke map with eSyncMate at the center, your ERP on the left, and the trading partner on the right. Shows data flow direction for each route.

notifications Notifications

When a route runs, you'll receive a notification with one of these statuses:

hourglass_topRunning

The route is currently executing. You'll see an update once it finishes.

check_circleCompleted

The route finished successfully. Data was exchanged without any issues.

errorFailed

Something went wrong during execution. Check the Exception Logs for details on what happened.
