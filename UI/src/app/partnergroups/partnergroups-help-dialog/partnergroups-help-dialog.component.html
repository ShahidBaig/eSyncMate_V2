touch_app What You Can Do

Use these actions to manage your partner groups:

add_circle
Add Partner Group

Create a new partner group by choosing a source party and a destination party from your customer list. The description is automatically filled in based on the party names you select.

edit
Edit Partner Group

Update the source party, destination party, or description of an existing partner group. Changes take effect the next time a route using this group runs.

settings Partner Group Fields

When adding or editing a partner group, you will work with these fields:

upload
Source Party The customer or system that sends data (for example, your ERP or a marketplace)
download
Destination Party The customer or system that receives data (for example, a marketplace or your ERP)
description
Description Automatically set to "Source Party - Destination Party" but you can change it if needed
