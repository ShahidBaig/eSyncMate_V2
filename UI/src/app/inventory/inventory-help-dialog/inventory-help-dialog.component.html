search How to Search Inventory Batches

To view inventory data, you must first select a customer. The grid shows Upload batches only:

1
Select ERP Customer ID Choose a marketplace partner from the searchable dropdown. This is required — no data loads until a customer is selected
2
Set Date Range (Optional) Start Date defaults to yesterday and Finish Date defaults to today. Adjust if needed
3
Click Search eSyncMate loads all Upload batches for that customer into the main grid
4
Expand an Upload Row Click any row to expand it and see the Download batches (Full Feed / Differential Feed) that occurred between that upload and the previous upload
lightbulb You must select a Customer before searching. You can also filter by Item ID or Status for more specific results.

download Expandable Row — Download Batches

Click any upload row to expand it. The expanded section shows all Download batches (Full Feed and Differential Feed) that occurred between the current upload and the previous upload:

1
Date Range Calculation The system automatically calculates the time window: from the previous upload's start time to the current upload's start time
2
Download Data Loaded All Full Feed and Differential Feed batches for the same customer within that time window are fetched and displayed in a nested table
3
Actions Available Each download row also has an Actions button — click it to view the item-level detail for that download batch
lightbulb The download table shows: Batch ID, Route Type (Full Feed or Differential Feed), Status, Items Count, Start/Finish Date, and Actions. If no downloads occurred between two uploads, a "No download data found" message is shown.

label Batch Statuses

Each batch (upload or download) has a color-coded status:

hourglass_topProcessing

When set: As soon as eSyncMate begins processing the batch.

What it means: The batch is currently running. Wait for it to finish.

check_circleCompleted

When set: After all items have been successfully processed.

What it means: The batch finished successfully. No action needed.

errorError

When set: When something went wrong during processing.

What to do: Check the Exceptions page for details.

cancelAborted

When set: When a higher-priority route (Full Feed) caused this batch to stop.

What it means: The batch was gracefully stopped. It will retry on the next schedule.

touch_app Show Inventory — Batch Detail Popup

Click the Actions icon button on any upload or download row to open the batch detail popup:

info Popup Header

Shows the Customer ID, batch Status (color-coded chip), and total item count.

search Search by Item ID

Use the search bar to filter items within the batch by Item ID.

view_column Item Data Columns

Item ID, Date, Status, Customer Item Code, ETA Date, ETA Qty, Total ATS, and individual warehouse ATS columns (L10, L21, L28, etc.).

lightbulb ATS means "Available to Sell" — the total quantity available across all warehouse locations. Each warehouse column (L10, L21, L28, etc.) shows stock at that specific location.
