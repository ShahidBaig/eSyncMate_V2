touch_app What You Can Do

Use these actions to manage your maps:

add_circle
Add Map

Create a new data transformation map. Enter a name, choose the map type, and write the transformation rules in the built-in editor.

edit
Edit Map

Update the map name, type, or transformation rules. Changes take effect the next time a route using this map runs.

settings Map Configuration

When adding or editing a map, you will work with these areas:

label Map Name

A clear, descriptive name for the transformation (for example, "Order to ERP" or "Inventory to Amazon").

category Map Type

The category of transformation. This determines which routes can use this map.

code Map Editor

The JSON-based editor where you define how source fields map to destination fields with any needed conversions.

lightbulb Maps use a JSON transformation format that maps source fields to destination fields. Each field mapping can include value conversions, default values, and conditional logic to handle different data scenarios.
